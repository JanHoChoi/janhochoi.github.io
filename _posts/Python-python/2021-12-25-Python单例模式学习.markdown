python的单例模式感觉有很多不同的写法，总结感受一下。

## 使用模块实现

```python
class Singleton(object):
	pass
	
g_instance = None
def GetInstance():
    global g_instance
	if g_instance is None:
		g_instance = Singleton()
	return g_instance
```
